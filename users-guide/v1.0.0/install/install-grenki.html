Install Grenki

Assumptions:

  • You’ve installed an active Node.js LTS release on your Linux machine.

On this page, you’ll learn:

  • How to run Grenki without installing it.

  • How to install Grenki locally (recommended).

  • How to install the Grenki CLI and site generator packages globally.

Try before you install

If you’re just evaluating Grenki, you can use npx to run it without having to install it first.
